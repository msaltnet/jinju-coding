전기와 반도체 그리고 프로그래밍에 대해서 알아보고, 직접 회로와 프로그램을 만들어보자.

## 도전 과제
- [멜로디 연주하기](./play-button) - 버튼을 누르면 멜로디가 연주되는 프로그램을 만들어보자

- 프로그래밍할 PC 준비

- 버튼으로 노래 연주하기로 테스트
